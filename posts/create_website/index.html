A guide to create my website

Saturday, June 5, 2021

Hugo and Github Page

Hugo is an open source static site generator. You can find many cool themes inside. With help of Hugo and Github Pages, it is easy to deploy a customized website. Let’s see how it works.

OS Version:

  • Windows 10
  • Hugo 0.83.1
  • Theme: Notepadium 2.6.2

STEP 1. Install Hugo

A guide video can be seen here

  1. Create a directory at C:/Hugo/bin
  2. Download hugo release
  3. Extract all to path C:/Hugo/bin
  4. Add C:/Hugo/bin to environmental path
  5. In your terminal, type hugo version to check if installed correctly

STEP 2. Install theme

Find your favourite theme, and download it!

STEP 3. Deploy theme

Open a new terminal:

cd C:/Hugo/
hugo new site first_site
cd first_site/themes
git clone https://github.com/cntrump/hugo-notepadium.git

Don’t forget to rename your theme to hugo-notepadium.

STEP 4. Edit config.toml

Modify your config.toml

with your github username and website title.

STEP 5. Create the first blog and test locally

cd first_site
hugo new about.md
hugo server

we should be able to see the website at http://localhost:1313/ now:

STEP 6. Deploy to Github Pages

Open anoter terminal:

cd first_site
hugo

Then you’ll see a public directory appears under first_site directory. Then we need to create the Github personal page:

Don’t include README file:

After that, go back to the terminal:

cd first_site/public
git init
git add .
git remote add origin https://github.com/username/username.github.io.git
git commit -m “first commit”
git push origin master

You can check your deployment by Github:

Or directly search https://username.github.io/ in your browser.
